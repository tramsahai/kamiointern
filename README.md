# KamioKash Wallet

A wallet homepage for the KamioKash rewards app.

## Features

- Wallet balance showing 1000 KashCoins 
- Welcome message with user name
- Send, Receive, and Scan buttons
- Recent activity section with transaction history

## Tech Stack

- Next.js
- Tailwind CSS
- TypeScript
